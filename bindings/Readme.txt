-----------------------------------------------------------------------
 ■ 概要

	Lumino はゲーム制作に必要な機能をまとめた開発支援ライブラリです。
	現在は音声機能のみを公開しています。

-----------------------------------------------------------------------
 ■ 動作環境

	OS				Windows XP/Vista/7/8 32/64Bit (DirectX 9.0c 以上)
	CPU				Pentium または互換性のあるCPU (2 Core以上推奨)
	メモリ			256 MB以上
	サウンドカード	XAudio2 互換

	本ライブラリを使用したプログラムを実行するためには、
	DirectX のエンドユーザーランタイムをインストールする必要があります。
	制作物を配布する場合は、この旨を配布物に含めることをお勧めします。

	エンドユーザーランタイムは以下のページからダウンロードできます。
	http://www.microsoft.com/ja-jp/download/details.aspx?id=35
