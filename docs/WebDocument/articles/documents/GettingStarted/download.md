ダウンロード
====================

パッケージは開発環境ごとに用意されており、各ターゲットプラットフォーム向けのビルドに必要なファイルが含まれています。

Windows パッケージには次のターゲットをビルドするためのファイルが含まれています。
- Win32
- Web
- Android

macOS パッケージには次のターゲットをビルドするためのファイルが含まれています。
- macOS
- iOS
