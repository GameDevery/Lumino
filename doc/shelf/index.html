Lumino

Lumino はゲームやインタラクティブアートなどのクリエイティブコーディングを支援するための、クロスプラットフォーム・アプリケーションフレームワークです。

Download 0.2.0-alpha View on GitHub

開発状況

ある程度APIの確定した機能から小出しにしていますが、現在ものすごく開発中です。実際にゲームを作りながら開発を進めており、一通り完成したら1.0.0をリリースする予定です。
このページに掲載したクラスなどのAPIは基本的に変更しない予定ですが、1.0.0リリースまでに実際のゲーム開発で出た問題や不都合による変更は積極的に行います。
