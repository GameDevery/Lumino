Lumino
========

Lumino はゲームやデジタルアートなどのリアルタイムグラフィックスアプリケーションを構築するためのフレームワークです。

マルチプラットフォームで動作し複数のプログラミング言語から利用できる、使いやすいインターフェイスを提供します。
