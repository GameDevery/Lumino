Lumino for .NET $(LuminoVersion)
====================

導入方法
--------------------
Visual Studio 2015 がインストールされている必要があります。
導入する前に、Visual Studio を起動している場合は終了してください。

1. Lumino_Install.bat を実行します。
2. Visual Studio 2015 を起動し、メニューから [ファイル] → [新しいプロジェクト] を選択します。
3. [テンプレート] → [Visual C#] の一覧にある [Lumino プロジェクト] から新しいプロジェクトを作成します。
4. 作成されたプロジェクトを ビルド・実行します。
