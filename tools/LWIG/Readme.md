

Metadata
--------------------
### LN_CLASS

#### Static
class 定義。static クラスを示す。


### LN_METHOD

#### Property
C# のプロパティ構文などへ変換することを示す。
関数名の先頭が Get, Set, Is の場合は除かれる。


class
--------------------
ベースクラスのテンプレートのネストはできない。
class A : public List<List<>> {}
public 継承以外使えない。


struct
--------------------
"Set" という名前の関数は使えない。


const メンバ変数
--------------------
constとして宣言されたメンバーは、読み取り専用メンバーとしてラップされ、定数は作成されません。
